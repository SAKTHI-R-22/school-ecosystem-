# School Ecosystem Management System

A full stack school ecosystem platform developed for managing communication and activities between students, parents, teachers, and administrators.

## Features

### Student
- View attendance
- View marks
- Apply leave
- Participate in events
- Receive notifications

### Teacher
- Assign homework and assignments
- Communicate with parents and students
- Manage attendance and marks

### Parent
- Communicate with teachers
- Receive school notifications
- Track student progress

### Admin
- Manage users
- Announce holidays and events
- Send notifications

---

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
